# Credit Card Fraud Detection

## Description
Ce projet a pour objectif de détecter les transactions frauduleuses sur des cartes bancaires à partir d'un jeu de données de transactions.  
Il illustre l'utilisation des techniques de **Data Science** pour l'analyse de données et la construction de modèles prédictifs.

## Technologies
- Python 3
- Pandas & NumPy
- Matplotlib & Seaborn (visualisation)
- Scikit-learn (modélisation)
- Jupyter Notebook

## Fichiers du projet
- `Credit_Card_Fraud_Detection.csv` : jeu de données des transactions
- `Detection de fraudes.ipynb` : notebook d'analyse, traitement et modélisation
- `README.md` : documentation du projet

## Étapes réalisées
1. Analyse exploratoire des données (EDA)
2. Pré-traitement et nettoyage des données
3. Séparation des données en train/test
4. Construction et entraînement d’un modèle de classification
5. Évaluation des performances (accuracy, recall, precision)
6. Visualisation des résultats et interprétation
